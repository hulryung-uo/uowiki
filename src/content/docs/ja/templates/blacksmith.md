> **ステータス: 未検証。** このストーリーは古典時代のコミュニティの知恵を我々のシャード(shard)向けに調整したものです。以下のスキル範囲とNPC価格はすべてサーバーソースと照合済みですが、ペース配分と稼ぎの見積もりは実プレイによる実地検証待ちです。

ブラックスミスは経済キャラクターです: 鉱石を掘り、製錬し、それを叩いて品物にし、他のプレイヤーの冒険を自分の収入に変えます。最終目標: **GM Blacksmithy + GM Mining**、マークした採掘スポットのルーンブック、安定した[バルクオーダー](#bulk-order-deeds)の習慣、そして「人々が壊れたプレートを持ち込む鍛冶屋」という評判。

## このシャードでのキャラクター作成

サーバーが強制する[作成ルール](/ja/mechanics/character-creation/)に従って(ステータスポイント90、スキル4つ×最大50、合計120):

| 選択 | 選ぶもの | 理由 |
|---|---|---|
| ステータス | **STR 60 / DEX 15 / INT 15** | STRは積載量 — 鉱石は重い。これはサーバー自身のBlacksmithプロフェッションテンプレートと正確に一致します。 |
| スキル | **Blacksmithy 50、Mining 50、Tinkering 20** | プロフェッションボタンではなく*カスタム*テンプレートを使うこと — カスタムの選択はスキルごとに初期装備を付与します。 |
| 都市 | **[Minoc(マイノック)](/ja/world/minoc/)** | 山、炉、金床、銀行が一つのまとまった動線に収まっています。 |

初期装備: Blacksmithyは**トング、つるはし2本、鉄インゴット50個、半エプロン**を付与。Miningは3本目のつるはしを追加。Tinkeringは**ティンカーツール + さらにインゴット50個**を追加。開始時の金貨1,000から、Minocの採掘者/ティンカーで購入を: 予備の**シャベル(12 gp)**と**つるはし(22〜25 gp)**、そして動物調教師から**パックホース(631 gp)**か**パックラマ(565 gp)**を検討 — 1回の遠征あたりの収穫がおおよそ倍になります(`Scripts/VendorInfo/SBMiner.cs`、`SBAnimalTrainer.cs`)。

**ここでは当てはまらない当時の助言:** 古典ガイドは鉱石をPKの前で運ぶ危険を警告します。あなたは**Trammel**で始まり、このシャードでレッドはFeluccaに制限されています(`Config/General.cfg`) — 加えて新規アカウントにはYoung保護があります。安心して採掘しましょう。

## ステージ1 — 初心者(Smithy 50→65、Mining 50→65)

**目標:** 自己資金で回るループ。[Minocの鉱山](/ja/world/minoc/)で鉄を採掘し、炉で製錬し、クラフトし、製品を製錬し戻し、余剰を銀行へ。

- **製錬はMiningチェックです**(`Scripts/Items/Resource/Ore.cs`): 鉄鉱石は25〜75のウィンドウで製錬され、*製錬に失敗すると対象の鉱石の山の半分が燃えます*。Mining 50では製錬の約半分に失敗します — Miningが約75になり鉄の製錬が失敗しなくなるまで、**小さな山で製錬**しましょう。
- **カットラスをクラフト** — 我々のシャードではカットラスのウィンドウは**24.3〜74.3**([クラフト表](/ja/crafting/blacksmithy/))なので、当時標準の「50〜65でその後切り替え」ではなく、50(≈64%成功)から約65まで運んでくれます。1本あたり8インゴット。
- クラフトgump経由で**トレーニングの産物を製錬**: 回収率は**インゴットの66%**(`Scripts/Services/Craft/Core/Resmelt.cs`)なので、カットラス1試行あたり約3インゴットの損失です。
- **「すべてクラフト」**のバッチを使いましょう — 1クリックあたり複数の上昇チェック([スキル上昇](/ja/mechanics/skill-gain/))。

今の稼ぎ: 正直、薄い。NPCの鍛冶屋は**鉄インゴット1個につき3 gp**を払います(棚価格5 gpの75%、`SBBlacksmith.cs` + `Scripts/VendorInfo/GenericSell.cs`) — 安い武器を売るより生インゴットの余剰を売るほうがましです(カットラスはわずか12 gpにしかならず、その8インゴットの生の価値より*低い*)。

## ステージ2 — 一人前(Smithy 65→85、Mining 65→85)

- **トレーニング表(我々の数値、[/crafting/blacksmithy/](/ja/crafting/blacksmithy/)より):** 約65から**クリス 36.7〜86.7**(8インゴット)、その後約75から**ショートスピア 45.3〜95.3**(6インゴット)。
- **Mining 65+で色付きの鉱脈が開きます** — 65でダルカッパー、70でシャドウ、75でカッパー、80でブロンズ([鉱石表](/ja/items/resources/))。色付きインゴットはNPCではなくプレイヤーの鍛冶屋に売れます。
- **Tinkering:** 当時の助言は「Tinkering 20でシャベルが作れる」と言いますが — **ここでは違います**。シャベルには**Tinkering 40〜90**が必要です([ティンカリング表](/ja/crafting/tinkering/))。**はさみ(5〜55、2インゴット)**で20→40を鍛え、その後シャベルを自作しましょう。それまでは12 gpのNPCシャベルで十分です。
- Mageryの仲間がいる(かRecallスクロールを買う)なら、**鉱山、炉、銀行でマークしたルーンブック**を手に入れましょう — 運搬ループこそがこのプロフェッション全体です。

## ステージ3 — 達人(Smithy 85→GM、Mining 85→GM)

- **プレートメイルゴージット 56.4〜106.4**(10インゴット)が約80→95を運び、**プレートメイルグローブ 58.9〜108.9**(12インゴット)がGMまで運びます — どちらもウィンドウがGMの先まで伸びているため、100.0*でも*なお上昇します。
- **プレートのアービトラージ:** NPCの防具商は**プレートゴージット1個につき52 gp**、**プレートグローブ1個につき72 gp**を払い(`SBBlacksmith.cs`)、エクセプショナルなら+25%。プレートグローブは12インゴットを72〜90 gpに変えます — **1インゴットあたり6+ gp、生インゴットレートの倍**です。あなたのGMトレーニングは自前で元が取れます。当時のガイドはGMまでに40〜55kインゴットを大半製錬し戻す前提で予算しましたが、ここでは高位品が*売れます*。
- Mining 85→99でゴールド/アガピート/ベライト/**ベラライト(99)**の鉱脈が開きます — プレイヤーのウォリアーが実際に金を払う名誉のインゴットです。

## バルクオーダーディード

スミスのBODはこのシャードで稼働しています(`Scripts/Services/BulkOrders/BulkOrderSystem.cs`、EJ下でTOL方式のシステムが有効): どのNPC鍛冶屋も**6時間ごとに1つ、最大2つまで貯められる**形で提供します(`Delay = 6`、`MaxCachedDeeds = 2`)、Blacksmithyのスキルがいくらかあれば。トレーニングの産物から鉄の小BODを埋め、銀行に寄るたびに集めましょう — 報酬にはNPCが売るどんなものよりも優れたスミスツールと材料が含まれます。

## 交易ループ

| 行動 | 場所 | 価格(ソース照合済み) |
|---|---|---|
| 生の鉄インゴットを売る | NPCの鍛冶屋、どこでも | 各3 gp。棚価格はベンダーが吸収する1,000ユニットごとに約1 gp下がる(経済システム、`GenericBuy.cs`) — 大量放出は街を分散させること |
| 鉄インゴットを大量に売る | プレイヤーのクラフター(フォーラムの取引掲示板) | 3〜5 gp — NPCの5 gp棚価格を下回れば、その代替を下回ったことになる |
| プレートゴージット/グローブを売る | NPCの防具商 | 52 / 72 gp固定(エクセプショナルは+25%)。経済アイテムではなく、価格下落なし |
| 色付きインゴット&エクセプショナルプレートを売る | プレイヤーのみ | NPCはベラライトに鉄と同じ額しか払わない — 色付きを決してNPC売りしないこと |
| 修理 | 賑わう炉(Britain、Minoc) | チップ。修理は後にあなたのプレートを買う顧客リストを築く |
| シャベル/つるはしを買う | NPCの採掘者/ティンカー | Tinkering 40+まで12 / 22〜25 gp |

[Britain(ブリテイン)](/ja/world/britain/)の銀行は、売り込む価値のある在庫があるときの王国のマーケットプレイスです。[Vesper(ベスパー)](/ja/world/vesper/)はMinocの小規模なベンダー陣容に欠けるものを補います。

## 判断ポイント & よくある失敗(エージェント向け)

- **常に過積載なら** → STRが低すぎるかパック動物がいない。歩く前に山腹の炉に隣接する地点で製錬し、インゴットは鉱石よりはるかに軽いことを覚えておくこと。
- **製錬で鉱石を破壊し続けるなら** → あなたのMiningが約75未満。小さな山で製錬するか、損失をMiningトレーニングと割り切ること(チェックはスキルを上げます)。
- **クラフトアイテムの成功率が約80%を超えたら** → 上昇が鈍りました。表を下げましょう(カットラス → クリス → ショートスピア → プレートゴージット → プレートグローブ)、[実際のウィンドウ](/ja/crafting/blacksmithy/)に従って。
- **トレーニング武器をNPC売りしたくなったら** → やめること。製錬はインゴットの66%を回収し、11〜16 gpの売却より価値があります。例外: プレートゴージット/グローブは金属価値を上回って売れます。
- **色付きの鉱脈が普通の鉄を産んだら** → あなたのMiningがその鉱石の要件を下回っています([表](/ja/skills/mining/))。鉱脈は無駄になっておらず、後で戻ってきましょう。
- **あるベンダーでインゴット価格が崩壊したら** → そこに約1,000+ユニットを食わせました。街をローテーションするか、プレイヤーのバルク市場に切り替えましょう。
- **「Blacksmith」プロフェッションボタンを選んだら** → 固定30ポイントのスキルとカスタム装備なしになります。ここではカスタムテンプレートが厳密に優れています。
- **AFKマクロを当てにしないこと** — マクロ対策の*コード*はオフですが([スキル上昇](/ja/mechanics/skill-gain/))、古典ガイドは無人プレイを#1のBANの磁石と呼びます。このシャードの明文規定はこれに触れていないので、許容されると決めつける前にフォーラムで尋ねましょう。
