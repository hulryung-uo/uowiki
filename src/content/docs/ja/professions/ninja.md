## このプロフェッションとは

ニンジャ(Ninja)は影から打撃を繰り出すステルス戦士です。[忍術(Ninjitsu)](/ja/skills/ninjitsu/)は
隠れることを武器に変えます — 奇襲ダメージ、おとり、短距離テレポート、そして移動と特殊能力のための
動物変身です。このシャードでは忍術は実装済みです(EJ 拡張スタックにはサムライ・エンパイア時代の
スキルが含まれます)。ヒットアンドフェイドの立ち回りのために、ステルス系デクサーやアーチャーと
組み合わさります。

## コアスキル

- [忍術(Ninjitsu)](/ja/skills/ninjitsu/) — 規律のスキル。その能力と変身を動かします。
- [隠蔽(Hiding)](/ja/skills/hiding/)と[ステルス(Stealth)](/ja/skills/stealth/) — いつでも姿を消し、隠れたまま移動します。奇襲と逃走が依存する基盤です。
- 武器スキル — [フェンシング(Fencing)](/ja/skills/fencing/)(鎌、釵)または[剣術(Swordsmanship)](/ja/skills/swordsmanship/)。アーチャーは代わりに忍術と弓を組み合わせます。
- ダメージ用の[戦術(Tactics)](/ja/skills/tactics/)と[解剖学(Anatomy)](/ja/skills/anatomy/)、包帯用の[治療(Healing)](/ja/skills/healing/)。

## ビルド

まだ**専用のニンジャ・テンプレートページはありません**。忍術枠を持つステルス系デクサーとして
組みましょう。戦闘の核については[ウォリアー・テンプレート](/ja/templates/warrior/)から始め、そこに
忍術 + 隠蔽 + ステルスを加えます。よくある構成は、武器 + 戦術 + 解剖学 + 忍術 + 隠蔽 +
ステルス + 治療です。700 ポイントの予算については[7x GM テンプレート](/ja/templates/seven-gm/)を
参照してください — 隠蔽 + ステルス + 忍術は3枠を費やす重い投資である点に注意しましょう。

## プレイの仕方

まず[隠蔽とステルス](/ja/playing/hiding-and-stealth/)を読みましょう — それが核心そのものです。次に
スイングのループと特殊技については[戦闘の基礎](/ja/playing/combat-basics/)と[戦闘応用](/ja/playing/combat-advanced/)を、
[魔法スクール](/ja/playing/magic-schools/)は忍術を各スクールの中に位置づけます。

ツールキット(すべて `servuo: Scripts/Spells/Ninjitsu/` 配下で確認済み):

- **アニマルフォーム(Animal Form)** — 生き物に変身して移動速度と変身固有の能力を得ます。ニンジャの代名詞的な機動力です。
- **ミラーイメージ(Mirror Image)** — 攻撃を吸収するおとりの分身を生み出します。
- **シャドウジャンプ(Shadowjump)** — 影から影への短いテレポートで、位置取りや逃走を行います。
- **サプライズアタック(Surprise Attack)** / **バックスタブ(Backstab)** — 隠れた状態から打撃したときのボーナスダメージです。
- **デスストライク(Death Strike)** — ターゲットが動くと罰を与える遅延型の負傷です。
- **キアタック(Ki Attack)** / **フォーカスアタック(Focus Attack)** — モメンタムとフォーカスに基づく命中ボーナスです。
- **スモークボム(煙玉)**(クラフト可能なアイテム)は戦闘中でも瞬時にあなたを隠します。**手裏剣(shuriken)**は投擲する遠隔の打撃です。

## 装備

- [武器(Weapons)](/ja/items/weapons/)と[武器カタログ](/ja/items/catalog/weapons/) — 速いフェンシング武器(鎌、釵)は奇襲バーストに合います。**手裏剣**と**スモークボム**を携帯しましょう。
- [防具(Armor)](/ja/items/armor/) — ステルスを妨げないほど軽く、バランスの取れた抵抗値。
- [マジックアイテムの特性](/ja/magic/) — Swing Speed、Hit Leech、そしてステルス/隠蔽と相性の良いモッド。**包帯**を携帯しましょう。

## 生計の立て方

ニンジャは奇襲で稼ぎます — 隠れた状態から大きな初撃を放ち、打撃の合間に姿を消して再び隠れることで
反撃ダメージを減らします。アニマルフォームは狩り場間の移動を速めます。ステルスキットは偵察や
PvP 回避にも向いています。戦利品は[ベンダーと銀行](/ja/playing/vendors-and-banking/)を通じて売りましょう。
