PvM の[プロフェッション](/ja/professions/)ハブは「テイマー/スミスに*なりたい*」という問いに
答えます。このページは別の問いに答えます。**「人は他のプレイヤーと戦うために何を構築するのか?」**
これらは戦闘の原型 — モンスターではなく他のキャラクターを倒すために調整されたテンプレートです。

## このシャードでの PvP

プレイヤー戦闘の**ルール**は[名声と PvP](/ja/playing/notoriety-and-pvp/)にあります。犯罪者と
殺人者のフラグ付け、ガードゾーン、そしてフェルッカ式のオープン PvP です。そのページは、誰かを
*いつ・どこで*攻撃できるか、そして攻撃したときに名声に何が起こるかを教えてくれます。

**この**ページは、それらの戦いに勝つために人々が構築する*キャラクター*についてです。オープン PvP が
利用できるかどうか、そしてフラグ付けが正確にどう機能するかは、シャードのルールによって支配されます。
まずそのページを読み、それからここに戻ってビルドを選んでください。

## PvP ビルドを特別にするもの

PvM の農夫は、予測可能なモンスターに対する生の持続的なダメージを最適化します。PvP ビルドは、
2秒のコンボであなたを殺そうとする思考する相手に対する**速度と生存性**を最適化します。優先順位が
変わります。

- **瞬間バースト** — 勝利とは通常、HP をじわじわ削ることではなく、敵が反応できるより速く致命的なコンボを叩き込むことです。
- **呪文中断** — ダメージを受けるとキャスターの詠唱が中断されるため、*相手*を中断させる打撃を当てることと、被弾*しながら*詠唱しきることの両方が重要になります。
- **脱出と機動力** — リコール、テレポート、そして走るためのスタミナ。離脱できる戦いは、負けようのない戦いです。
- **圧力下での解毒と回復** — 包帯、キュア、グレーター・ヒールで毒のダメージと受けるダメージを生き延びること。
- **コンボで死なないこと** — 開幕の一撃を生き延びるだけの十分な HP、抵抗、そして反応的な回復。

このため、[魔法抵抗(Resisting Spells)](/ja/skills/resisting-spells/)(敵の魔法を受け流すため)、
[レスリング(Wrestling)](/ja/skills/wrestling/)(近接されたメイジでも詠唱でき、自動的に中断されない
ため)、そして素早い[包帯回復](/ja/playing/healing/)は、PvM よりも PvP ではるかに重要になります。

## 古典的なビルドの原型

これらは 700 ポイント上限内のテンプレートです([7x GM テンプレート](/ja/templates/seven-gm/)を参照)。
この上限こそが、以下の難しい選択を強いるものです。

### タンクメイジ

**スキル:** 魔術、知力評価、魔法抵抗、レスリング、解剖学、治療、瞑想。

万能型であり**PvP テンプレートの基準点**です。攻撃魔法を詠唱し、ダメージを受けながら**包帯で回復**し
(解剖学 + 治療)、近接に詠唱をロックされないよう**レスリング**します。何でもこなす器用貧乏ならぬ
器用万能 — 倒しにくく、手詰まりにならず、目立った弱点がない — だからこそ、他のすべてがそれと
比較される基準のテンプレートなのです。[メイジ](/ja/professions/mage/)を参照。

### ノックス(毒)メイジ

**スキル:** 魔術、毒(Poisoning)、知力評価、魔法抵抗、レスリング、瞑想、+ 少しの治療。

持続力の一部を[毒(Poisoning)](/ja/skills/poisoning/)と引き換えにしたタンクメイジです。
**毒 + エクスプロージョン/エナジーボルト**のコンボで口火を切ります。毒のダメージが敵の回復に
圧力をかける一方でバーストが着弾し、毒を受けた相手は反撃する代わりに解毒に行動を費やさなければ
なりません。攻撃は強力ですが、純粋なタンクメイジよりも自前の回復は薄めです。

### スタンメイジ

**スキル:** 魔術、レスリング、魔法抵抗(時代依存の追加要素)。

**スタンとロックダウン**の技 — 古典的なレスリングスタン、または武器破壊 + パラライズのコンボ —
を使って、キルコンボが着弾する間ターゲットを凍りつかせます。成功すれば被害者は反応できないため
壊滅的です。

:::caution[当シャードでは未検証のメカニクス]
レスリングスタンとパラコンボのロックは**強く時代依存**です(AOS 以前のレスリングスタンは、後の
拡張とは大きく異なる挙動をします)。当 EJ シャードでの正確なメカニクスは**未検証**です — サーバー
エミュレーターのソースとゲーム内テストで確認されるまで、この原型は確定したレシピではなく概念として
扱ってください。
:::

### ピュア / 5x メイジ(スクライブメイジ)

**スキル:** 魔術、知力評価、魔法抵抗、瞑想、レスリング + 写本。

**ガラスの大砲**です。[写本(Inscription)](/ja/professions/scribe/)(スクライブ)は呪文ダメージを
底上げし、このビルドが消費していくスクロールを供給します。デクサーの包帯回復を、最大限の魔法
バーストと引き換えにします。タンクメイジより強く打ちますが、誤りの余地は少なくなります —
開幕で勝つか、よりタフな相手を持ちこたえるのに苦労するかです。[スクライブ](/ja/professions/scribe/)を参照。

### デクサー(近接 PvP)

**スキル:** 武器スキル(剣術 / フェンシング / メイス)、戦術、解剖学、治療、魔法抵抗、受け流し —
多くはリコール用の魔術を少し加えます。

近接戦士です。**追って、バースト。** 距離を詰め、武器破壊 / 脳震盪打撃 / 麻痺打撃の特殊技を当てて
ターゲットをロックし、武器ダメージと包帯で削り倒します。[受け流し(Parrying)](/ja/skills/parrying/)と
[魔法抵抗(Resisting Spells)](/ja/skills/resisting-spells/)がメイジに対して生かしてくれます。少しの
魔術はもっぱらリコールの機動力のためです。このビルドが頼りにする特殊技については
[ウォリアー](/ja/professions/warrior/)と[戦闘応用](/ja/playing/combat-advanced/)を参照。

### ステルスアーチャー / ニンジャ

**スキル:** アーチェリー(またはニンジツ)、ハイディング、ステルス、戦術、解剖学、治療またはシバルリー。

隠密状態から**強烈なアルファストライク**で口火を切り、その後位置取り直して再び隠れリセットします。
ゲームプレイは待ち伏せして消えること — 大きな開幕の一撃を与え、視線を切り、ターゲットが反撃できる
前に姿を消します。[ニンジャ](/ja/professions/ninja/)、[アーチャー](/ja/professions/archer/)、
そして[ハイディングとステルス](/ja/playing/hiding-and-stealth/)を参照。

### バード / ディスコメイジ

**スキル:** 不協和音 + 挑発(そして演奏)を、メイジのコアに重ねたもの。

キルの前にターゲットのステータスとスキルを弱体化させる**[不協和音(Discordance)](/ja/skills/discordance/)**
と、コントロール用の挑発を加えたメイジです。歌による弱体化が敵を柔らかくし、魔法がより速く
仕留めます。[バード](/ja/professions/bard/)を参照。

### サンパイア式近接 / ネクロデクサー

**スキル:** 吸血リーチ付きの武器スキル + 生存性のためのブシドーおよび/またはネクロマンシー。

グループ戦では、**武器の生命吸収**(ヴァンパイリック・エンブレイス / ヒット・ライフ・リーチ)に
ブシドーまたはネクロマンシーを加えることで、普通のデクサーなら倒れる場面で近接戦士を立たせ続けます。
決闘者というよりは、乱戦のための耐久力あるブルーザーです。[サンパイア](/ja/professions/sampire/)を参照。

### シーフ(PvP)

**スキル:** スティーリング、ハイディング、ステルス(多くは脱出用に少しの戦闘または魔術)。

専門職です。**[スティーリング(Stealing)](/ja/skills/stealing/)**を使って、戦いの混乱の中から
アイテムをかすめ取る — あるいは武器を奪う — そして[ハイディングとステルス](/ja/playing/hiding-and-stealth/)
で戦利品とともに姿を消します。誰も自分のパックに伸びる手に気づかない、混雑した PvP で真価を
発揮します。[シーフ](/ja/professions/thief/)を参照。

## PvP を定義づけるスキル

- [魔法抵抗(Resisting Spells)](/ja/skills/resisting-spells/) — 敵の魔法の効果を軽減します。ほぼ必須。
- [レスリング(Wrestling)](/ja/skills/wrestling/) — 近接されたキャスターが詠唱を続けられるようにします(そしてスタンビルドを駆動します)。
- [治療(Healing)](/ja/skills/healing/) + [解剖学(Anatomy)](/ja/skills/anatomy/) — 包帯回復、あなたの反応的な命綱。
- [魔術(Magery)](/ja/skills/magery/) — バースト、キュア、グレーター・ヒール、そしてリコールの機動力。
- [毒(Poisoning)](/ja/skills/poisoning/) — 敵の回復に圧力をかけます。
- [戦術(Tactics)](/ja/skills/tactics/) — あらゆるデクサーにとっての武器ダメージ倍率。
- [受け流し(Parrying)](/ja/skills/parrying/) — 打撃をブロックし、集中砲火を生き延びます。
- [瞑想(Meditation)](/ja/skills/meditation/) — 交戦の合間にマナを回復します。

## テンプレートについての注記

上記のすべてのビルドは**700 ポイント上限**に押し込められています — その予算がなぜトレードオフを
強いるのか(解剖学を外して写本を、治療を外して毒を、といった具合)については
[7x GM テンプレート](/ja/templates/seven-gm/)を参照してください。これらのビルドの PvM 版については、
[プロフェッション](/ja/professions/)ハブと比較してください。
